# CryoACE Project Page

Static GitHub Pages site for the CryoACE paper.

## Local preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Publish with GitHub Pages

1. Create a GitHub repository, for example `cryoace-page`.
2. Push this directory to the repository.
3. In GitHub, open `Settings -> Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Select branch `main` and folder `/root`.

The page will be available at:

```text
https://YOUR_USERNAME.github.io/cryoace-page/
```

Update the `arXiv soon`, `Code soon`, and `Data soon` links in `index.html` once those URLs are ready.
